# Blank Area

<div class="ref-description section level2">

Use this function to capture a click on a blank area of the canvas. Note
that this may stops other "click" events from working.

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
e_get_zr(e)
```

</div>

</div>

<div class="section level2">

## Arguments

-   e:

    An `echarts4r` object as returned by `e_charts` or a proxy as
    returned by `echarts4rProxy`.

</div>

</div>
