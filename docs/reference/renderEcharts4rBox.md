# Render box

<div class="ref-description section level2">

Render an echarts4r box.

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
renderEcharts4rBox(expr, env = parent.frame(), quoted = FALSE)
```

</div>

</div>

<div class="section level2">

## Arguments

-   expr:

    An expression that produces as `echarts4rBox`.

-   env:

    The environment in which to evaluate ‘expr’.

-   quoted:

    Is ‘expr’ a quoted expression (with ‘quote()’)? This is useful if
    you want to save an expression in a variable.
