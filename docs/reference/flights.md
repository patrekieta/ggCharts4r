## Format

A data frame with 178 rows and 8 variables:

-   start_lat:

    Numeric vector

-   start_lon:

    Numeric vector

-   end_lat:

    Numeric vector

-   end_lon:

    Numeric vector

-   airline:

    Character vector

-   airport1:

    Character vector

-   airport2:

    Character vector

-   cnt:

    Integer vector
